# 用户行为分析平台 · Demo

面向出行业务（顺风车/打车）的用户行为分析平台原型，基于埋点数据接入，覆盖页面流转、单页面热力、用户路径回放等分析能力。Demo 数据为模拟值，仅用于演示交互与设计稿。

## 在线预览

直接用浏览器打开：

```
app/userpath_demo.html
```

## 功能概览

平台包含三个分析视角（顶部 Tab）：

1. **页面流分析** — 页面之间的桑吉图流转、流失分析、节点下钻。
2. **核心页面分析** — 单页面 KPI、热力图、模块数据表、上下游来源、模块操作步骤桑吉图、版本对比。
3. **用户分析** — 分群高频路径检索、单用户路径时间线回放。

完整需求和数据口径见 [用户行为分析平台_需求文档.md](./用户行为分析平台_需求文档.md)。

## 技术栈

- 纯静态 HTML + CSS + JS（无构建步骤）
- [Apache ECharts](https://echarts.apache.org/) 5.4.3 渲染桑吉图
- 截图素材为产品提供的真实页面截图

## 本地运行

无需安装依赖，直接打开 HTML 即可。如需通过 HTTP 访问（避免某些浏览器对 `file://` 的限制）：

```bash
cd app
python3 -m http.server 8000
# 然后访问 http://localhost:8000/userpath_demo.html
```
