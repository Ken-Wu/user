v1.0 · 基于埋点数据

产品定位

面向出行业务（顺风车 / 打车）的用户行为分析平台，基于埋点数据接入，帮助产品、运营、数据同学定位用户在页面流转、页面内模块点击、用户路径中的行为特征与转化漏斗，支撑异动定位、改版评估、ABTest 复盘。

①

页面流分析

看用户在页面之间怎么走、在哪一步终止
查询配置
查询设置 起始 / 终止页面（正推 / 倒推） 会话类型：时间分割（默认 40 分钟）/ 订单分割 时间范围（开始日期 → 截止日期）
筛选设置 平铺 9 项：平台来源 / 同跨城 / 里程段 / 预约-实时 / 出发时间段 / 品类 / 是否同呼 / 乘客-车主 / 新老客
页面流转桑吉图
展示控制 每步 Top（3-10），其余归"更多分组" 展示步数（2-10），按步数自适应横向滚动
节点 hover 页面总 PV / UV、流转 PV / UV、终止量 / 终止率
节点点击浮层 页面缩略图 + 关键指标 高亮主干路径（向前/向后 PV 最大） 查看页面详情 → 下钻到「核心页面分析」并带入页面

核心页面分析

看单个页面内各模块的曝光、点击与操作流转
查询配置
查询设置 页面选择（默认从上层带入，如：询价页） 时间范围（开始 → 截止）
筛选设置（与页面流分析一致）
页面数据 · 整体指标
KPI 指标 页面总 PV / UV、页面总点击次数 人均浏览量、平均停留时长 跳出率、跳转率
流量上下游 上一层页面来源（PV / UV / 占比） 后续流量去向（PV / UV / 占比）
页面热力图
区块热力 + 数据气泡 颜色深浅 = 点击占比，气泡显示模块名与数值 工具：放大查看、显隐热力、显隐气泡
区块 ↔ 模块表 双向联动 hover 红色描边、click 蓝色高亮 + 自动滚动
版本对比 电梯询价页 vs 品类中心询价页（左右并排）
模块数据卡（Tab 切换）
Tab 1 · 模块数据 列：模块名 / 曝光 PV·UV / 点击 PV·UV / 点击率 / 点击占比 / 操作 点击率 = 点击UV / 曝光UV；表头点击排序 品类列表-组「查看」→ 弹窗：各品类明细 + 发单选择率
Tab 2 · 行为数据 品类列表 / 地图：滑动次数 + 纵-横向深度 电梯 1段 / 2段 / 3段 滑动次数
模块操作步骤桑吉图（按需展示）
点模块后向后展开 起始节点：进入页面 / 任一模块 会话类型、每步 Top、展示步数（同页面流分析）
